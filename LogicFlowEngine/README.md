# LogicFlowEngine

LogicFlowEngine is a .NET Standard 2.1 library for building, executing, and managing logic flows and node-based graphs. It provides a flexible, extensible engine for defining custom nodes, connecting them, and executing complex workflows programmatically.

## Features

- Node-based logic flow engine
- Custom node support (inherit and extend)
- Graph execution and traversal
- Serialization and deserialization of logic graphs
- Designed for integration with UI editors (such as Blazor/WebAssembly frontends)
- .NET Standard 2.1 compatible (usable in .NET Core, .NET, Xamarin, Unity, etc.)

## Getting Started

1. Add a reference to the LogicFlowEngine project or NuGet package.
2. Define your custom nodes by inheriting from the base node classes.
3. Build your logic graph by connecting nodes.
4. Execute the graph using the provided engine APIs.
